Ecommerce Website (Spring Boot)

## 📌 Description

This project is a backend system for an ecommerce website built with Spring Boot.
It provides APIs for managing products, users, shopping cart, and orders.

The goal of this project is to practice building a real-world scalable backend application following clean architecture principles.

---

## 🚀 Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven
* Lombok

---

## ✨ Features

### 👤 User

* Register / Login (JWT - planned)
* View products
* Add to cart
* Place orders

### 🛍️ Product

* Create / Update / Delete product (Admin)
* View product list
* Search products

### 🛒 Cart

* Add / remove items
* Update quantity

### 📦 Order

* Create order
* View order history

---

## 🏗️ Project Structure

* Controller → Handle API requests
* Service → Business logic
* Repository → Database access
* Entity → Database models

---

## ⚙️ How to Run

1. Clone repository
2. Configure database in `application.properties`
3. Run project:

```bash
mvn spring-boot:run
```
